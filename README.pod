=head1 App::Git::Workflow::Command::Cat

Adds a simple way of showing files stored in git (simpler that git-show at least)

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
